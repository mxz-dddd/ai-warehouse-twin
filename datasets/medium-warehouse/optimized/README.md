# Medium Warehouse Optimized Slotting

This directory contains the A5a R3 Phase-1 demo ABC slotting output for
`datasets/medium-warehouse/scenario.json`.

`abc-slotting-config.json` is generated by a deterministic demo heuristic:

- SKU demand is ranked by total ordered quantity across outbound and each-pick
  orders, then by order line count, then by SKU id.
- Candidate pick locations are ranked by modeled distance to the A-zone pick
  path anchor `aisle-a-main`, using the existing layout graph and shortest-path
  routing.
- Assignments pair the highest-ranked SKUs with the highest-ranked candidate
  pick locations.

This is a deterministic demo ABC slotting heuristic for R3 Phase-1. It is not a
globally optimal slotting solution. It does not include A/B comparison,
ComparisonArtifact output, `improvement_pct`, or optimization claims.

The baseline scenario and A4b baseline golden artifacts remain unchanged.
